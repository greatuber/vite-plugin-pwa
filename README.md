## Usage

Install

```bash
npm i vite-plugin-pwa -D # yarn add vite-plugin-pwa -D
```

Add it to `vite.config.js`

```ts
// vite.config.js
import PWA from 'vite-plugin-pwa'

export default {
  plugins: [
    PWA()
  ]
}
```
